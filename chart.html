RESTful Routes

A table of all 7 RESTful Routes

Name Path HTTP Verb Purpose Mongoose Method
INDEX /blogs GET Index routes - show all blogs Blogs.find({})
NEW /blogs/new GET show form to add new blog N/A
CREATE /blogs POST create new blog then redirect Blogs.create()
SHOW /blogs/:id GET show information of a particular blog Blogs.findById()
EDIT /blogs/:id/edit GET show edit form to edit a particular blog Blogs.findById()
UPDATE /blogs/:id PUT update the edited blog and then redirect Blogs.findByIdAndUpdate()
DESTROY /blogs/:id DELETE delete a particular blog and then redirect Blogs.findByIdAndRemove()
